## 为什么要做 Seater

### 让窗口管理变得直观有趣

传统的窗口管理工具往往冰冷机械，只是简单地把窗口排列整齐。

但我想要的不只是整齐，而是一种更直观、更有温度的方式来组织我的工作空间。

于是我创造了「工位」这个概念：**每个应用都有自己的专属座位。**

你可以给工位起名字、设置头像、添加待办事项，甚至写下工作说明。

窗口不再只是冷冰冰的矩形，而是有了归属感。

这种拟人化的设计，让窗口管理变成了一件有趣的事。

### 拖拽即绑定，自动吸附

使用 Seater 非常简单：把窗口拖到工位上，松手即可。

窗口会**自动吸附并填充整个工位区域，**无需手动调整大小和位置。

支持多显示器、多办公室场景，轻松切换不同的工作布局。

下次启动时，Seater 会自动恢复之前的绑定关系，一切都在原位。
